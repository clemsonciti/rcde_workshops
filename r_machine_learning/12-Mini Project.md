## 15. MINI PROJECT

## Description of Mini Project:

- This mini project evaluates your ability to use caret and machine learning, using a sample data science project, from scratch.
- The project is about supervised machine learning only
- It requires you to download data, clean data, split into training/testing, apply any machine learning model, and analyze the output.
